Using the Safari Theme Maker

Types : Click to include all Pokémon with that type, or right-click to exclude. If a Pokémon has their types both included and excluded, they will be out.

Pokémon : Click a Pokémon icon to include them into the theme, or right-click to exclude them. This will always have priority over Included/Excluded Types. Legendaries and Alternative Forms will only spawn if included by this method.

BST Range : Define a value for minimum (recommended 260~380) and maximum (recommended 480~600). Pokémon with BST below the minimum will always have maximum chance of spawning, while those with BST above will spawn more rarely as their BST go up. Pokémon with BST higher than or equal the maximum will not spawn at all.

Custom BST : Middle-click a Pokémon icon to define a custom BST (they will count as having this BST for this theme).

Icon : Define the number of a Pokémon to be the icon that shows when the theme is started.

Edit Rules : Define the contest rules for this theme.

IMPORTANT : To see which Pokémon can actually spawn on your theme, check the Pokémon icons with green background. It doesn't matter if they are listed anywhere else, if the icon is not highlighted then they will not spawn.
