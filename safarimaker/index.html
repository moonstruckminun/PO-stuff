Using the Safari Theme Maker

Types: Click to include all Pokémon with that type, or right-click to exclude. If a Pokémon has their types both included and excluded, they will be out.

Pokémon: Click a Pokémon icon to include them into the theme, or right-click to exclude them. This will always have priority over Included/Excluded Types. Alternative Forms will only spawn if included by this method.

BST Range: Define a value for minimum (recommended 260~380) and maximum (recommended 450~600). Minimum ensures Pokémon below this BST will always have a high chance of spawning, while Maximum makes it so only Pokémon below this value will spawn. Click the 'Apply BST' button after changing this.

Custom BST: Middle-click a Pokémon icon to define a custom BST (they will count as having this BST for this theme).

Icon: Define the number of a Pokémon to be the icon that shows when the theme is started.

Edit Rules: Define the contest rules for this theme.
